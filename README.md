

# Triangle Checker

##### Triangle Checker Project, 9/8/2015

#### By Kenny Grage & Jason Creighton

## Description

This app will take three inputs as the sides of a triangle and verify that it is a triangle,
then return what type of triangle it is (equilateral, isosceles, or scalene).

## Setup

- clone this repository
- navigate web browser triangle-tracker.html


## Technologies Used

HTML/CSS, Javascript, jQuery, Mocha, Chai, Bootstrap
